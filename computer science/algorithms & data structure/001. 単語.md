# 専門単語

|単語|意味|
|----|----|
|Full Table Scan|レコードを先頭から順番に見て探す方法|
|Index Scan|インデックスを用いた方法|
|BST|Binary Search Tree, 二分探索木|
|B+Tree|1つのノードがm個 (m>=2) の子ノードを持つことができる平衡木構造|
|DDoS|ディードースアタック|
