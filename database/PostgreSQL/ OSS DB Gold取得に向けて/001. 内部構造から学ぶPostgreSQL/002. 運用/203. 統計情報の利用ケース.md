## pg_class = オブジェクトの物理情報
- オブジェクト is
  - テーブル(relation)
  - インデックス
  - PK,FK制約 

```sql
SELECT
  oid,
  relname,
  relfilenode,
  relpages,
  reltuples
FROM
  pg_class
WHERE
  relname NOT LIKE '%pg_%';
```
