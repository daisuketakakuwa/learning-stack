# デジタル署名 IS A 電子署名
デジタル署名は、電子署名の一種である。
![](https://storage.googleapis.com/zenn-user-upload/1bc79629a65d-20220717.png)

|単語|説明|
|----|----|
|電子署名|文書の真正性(正しいこと/改ざんされていないこと)を証明するために用いられる技術の総称。|
|デジタル署名|電子署名の１つ＝**改ざんされていないことを証明する技術の１つ。**<br>「**公開鍵暗号方式**」を利用して文書の真正性を証明する。|
