> org.springframework.util.MultiValueMap

```java
//同じキーに2つ追加する。
MultiValueMap<String, String> params1 = new LinkedMultiValueMap<String, String>();
params1.add("a", "あ");
params1.add("a", "い");

//1つ目の値だけ取得する
String val = params1.getFirst("a");  //⇒"あ"

//あるキーの値をすべて取得する
List<String> list = params1.get("a"); //⇒"あ", "い"
```
